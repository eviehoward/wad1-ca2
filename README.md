# Assignment 1 Lego Collection App

This App is used to record and display Lego collections.

The collections are organised by Lego theme, with an additional Wishlist collection.

Each collection stores:
- an ID,
- the theme's title,
- an array of sets,
- an image of the theme's logo.

For each set, the following data is recorded:
- the set code,
- name,
- number of pieces,
- RRP (in euro).

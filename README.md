# Assignment 2 Lego Collection App

This App is used to record and display Lego collections. The collections are organised by Lego theme. Users can login
and manage their own collection.

Each collection stores:

-   an ID,
-   the theme's title,
-   an array of sets,
-   an image of the theme's logo.

For each set, the following data is recorded:

-   the set code,
-   name,
-   number of pieces,
-   RRP (in euro),
-   an image.

Collections/sets can be added, deleted, and edited. Statistics about the global collection and user's collection can be
viewed on the Statistics page.
